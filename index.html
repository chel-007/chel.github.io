New: Get Started on Machine Learning

Python Machine Learning 2nd Edition is a book that has helped me a lot aiming to demystify the whole topic of Maching Learning.Written by Sebastian Raschka & Vahid Mirjalili, the book provides hands-on examples on the concepts of ML and introduces the basic mathematics of ML in a simplified approach. I recommend this book to new-comers of Ml and also those that has seen one or two Course videos from platforms like Coursera and Edx, because this book gives more details on what you can learn from those courses.

This book covers:

  1. The golden rules of ML
  2. How to perform Logistic Regression on Six Feature Vectors
  3. Technical use of Python in ML
  4. Introduction to Tensorflow Framework
  5. Introduction to Scikit-Learn

and much more.
You can purchaes this book in Kindle format or as a paperback directly on Amazon
